🍽️ 簡単レシピ

河内晩柑のさっぱりサラダ

🕐 10分 / 2人分 — 河内晩柑
河内晩柑 1個ベビーリーフ 1袋生ハム 4枚オリーブオイル 大さじ1塩・こしょう 少々
  1. 河内晩柑は皮をむき、薄皮から果肉を取り出す
  2. 器にベビーリーフを敷き、果肉と生ハムを盛り付ける
  3. オリーブオイルを回しかけ、塩・こしょうで味を調える

あまなつかんのはちみつマリネ

🕐 15分 / 2人分 — あまなつかん
あまなつかん 1個はちみつ 大さじ2レモン汁 小さじ1ミントの葉 少々
  1. あまなつかんは皮と薄皮をむき、果肉を一口大にほぐす
  2. ボウルではちみつとレモン汁を混ぜ、果肉を加えて和える
  3. 冷蔵庫で10分冷やし、ミントを添えて完成

簡単りんごのバター焼き

🕐 15分 / 2人分 — りんご
りんご 1個バター 20g砂糖 大さじ2シナモン 少々
  1. りんごは芯を取り8等分のくし切りにする
  2. フライパンにバターを熱し、りんごを並べて両面をこんがり焼く
  3. 砂糖を加えて絡め、仕上げにシナモンを振る
