キャベツ

キャベツとツナの無限サラダ

👥 2人分
キャベツ 1/4個ツナ缶 1缶ごま油 大さじ1鶏がらスープの素 小さじ1白ごま 適量
  1. キャベツを千切りにし、塩少々(分量外)で揉んで水気を絞る
  2. 油を切ったツナ、ごま油、鶏がらスープの素を加えて和える
  3. 仕上げに白ごまを振る

かぶ

かぶのまるごとポタージュ

👥 2人分
かぶ 2個玉ねぎ 1/4個牛乳 200mlコンソメ 小さじ1バター 10g塩こしょう 少々
  1. かぶと玉ねぎを薄切りにし、バターで炒める
  2. 水100mlとコンソメを加え、柔らかくなるまで煮る
  3. ミキサーにかけ、牛乳を加えて温め、塩こしょうで味を調える

ほうれんそうのごま和え

👥 2人分
ほうれんそう 1束すりごま 大さじ2醤油 大さじ1砂糖 小さじ1

  2. すりごま、醤油、砂糖を混ぜ合わせる
  3. ほうれんそうと和えて器に盛る
