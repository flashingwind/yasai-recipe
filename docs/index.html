春キャベツのレモン塩昆布あえ

👥 2人分
キャベツ 1/4個塩昆布 大さじ1レモン汁 小さじ2ごま油 小さじ1白ごま 適量
  1. キャベツを一口大にちぎり、ポリ袋に入れる。
  2. 塩昆布、レモン汁、ごま油を加えて袋の上から軽く揉む。
  3. 5分ほどなじませ、器に盛って白ごまを振る。

なす

とろとろ焼きなすのだし浸し

👥 2人分
なす 3本めんつゆ(2倍濃縮) 大さじ3水 大さじ3しょうが すりおろし 小さじ1かつお節 適量
  1. なすはヘタを取り、縦に数本切り込みを入れる。
  2. グリルまたは魚焼き器で皮が真っ黒になるまで焼く。
  3. 皮をむいて食べやすく裂き、めんつゆ・水・しょうがを合わせたつゆに浸す。
  4. 冷蔵庫で10分ほど冷やし、かつお節をのせる。
🕐 10分
こまつな

こまつなと油揚げのさっと煮

👥 2人分
こまつな 1束油揚げ 1枚だし汁 200mlしょうゆ 大さじ1みりん 大さじ1
  1. こまつなは4cm長さに切り、油揚げは熱湯をかけて短冊切りにする。
  2. 鍋にだし汁・しょうゆ・みりんを入れて煮立て、油揚げを加える。
  3. こまつなの茎、続いて葉を入れ、1〜2分煮て火を止める。
