🍽️ 簡単レシピ

いちごとヨーグルトの簡単ムース

🕐 15分 / 2人分 — いちご
いちご 150gプレーンヨーグルト 200g生クリーム 100ml砂糖 大さじ3粉ゼラチン 5g水 大さじ2
  1. ゼラチンを水でふやかし、電子レンジで10秒温めて溶かす
  2. いちごをフォークで潰し、砂糖・ヨーグルトと混ぜる
  3. 生クリームを泡立てて加え、ゼラチン液を混ぜ込む
  4. 器に注ぎ冷蔵庫で2時間冷やし固める
  5. 飾り用いちごをトッピングして完成

フライパンで作るりんごのキャラメル煮

🕐 15分 / 2人分 — りんご
りんご 1個バター 20g砂糖 大さじ3レモン汁 小さじ1シナモン 少々
  1. りんごは皮付きのまま8等分のくし切りにする
  2. フライパンにバターと砂糖を入れ中火で溶かす
  3. りんごを加え両面をじっくり焼き色がつくまで焼く
  4. レモン汁を加え全体を絡める
  5. 器に盛りシナモンを振ってアイスを添えても美味

河内晩柑と生ハムの爽やかサラダ

🕐 10分 / 2人分 — 河内晩柑
河内晩柑 1個生ハム 50gベビーリーフ 1袋オリーブオイル 大さじ2塩こしょう 少々はちみつ 小さじ1
  1. 河内晩柑は皮をむき薄皮から実を取り出す
  2. 皿にベビーリーフを広げ生ハムと河内晩柑を盛る
  3. オリーブオイル・はちみつ・塩こしょうを混ぜドレッシングを作る
  4. 食べる直前にドレッシングを回しかけて完成
